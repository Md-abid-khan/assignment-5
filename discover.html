Question-1: What are the different ways to select an element in the DOM?

Ans: There are many ways to select an element in DOM.

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. getElementsByName()
  5. querySelectorAll()
  6. querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:
  1. innerHTML return the HTML of the selected element
  2. innerText return the text of the selected element.
  3. textContent return the text of selected element and it's child
